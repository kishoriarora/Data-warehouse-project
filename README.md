# Data-warehouse-project 
Welcome to the Data Warehouse and Analytics Projet repository!🚀
This project demonstrate a comprehensive data warehouse and analytics solution, form building a data warehouse to generating actionable insights.


🏗️Date Architecture:
The data architecture for thsi project follows Medallion Architecture Bronze, Silver, Gold layers:

Bronze Layer: Stores raw data as-is from the source systems. Data is ingested from CSV Files into SQL Server Database.
Silver Layer: This layer includes data cleansing, standardization, and normalization processes to prepare data for analysis.
Gold Layer: Houses business-ready data modeled into a star schema required for reporting and analytics.
